## 🌌 O Mapa Estelar de Anápolis

No dia **20 de junho de 2025** — primeiro dia do inverno austral — o céu de Anápolis, GO (16°33'S, 48°57'W) mostrava este espetáculo:

| Constelação | Posição no céu | Curiosidade |
|------------|---------------|-------------|
| 🦂 **Escorpião** | Quase no zênite! | Antares brilhava em vermelho intenso |
| ✝️ **Cruzeiro do Sul** | Sul-sudeste | Sempre visível do hemisfério sul |
| 🐾 **Centauro** | Sul | Alpha Centauri, a mais próxima do Sol |
| 🏹 **Sagitário** | Leste | Apontando para o centro da Galáxia |
| 🐕 **Cão Maior** | Oeste | Sirius, a mais brilhante do céu noturno |

> 💡 **Clique nas estrelas** do mapa para descobrir seus nomes, magnitudes e mitologias!

## 🎮 Os Jogos

### 🔤 Wordle — Palavra Secreta
- Palavra de **4 letras** relacionada ao amor de Kauê por Gyovanna
- 6 tentativas, feedback por cores (verde = certo, amarelo = errado lugar, cinza = não existe)
- Teclado virtual + teclado físico suportado
- Animação de flip 3D nas tiles

### 🃏 Jogo da Memória
- **16 cartas**, 8 pares temáticos (❤️💕⭐🌙🎵🌹💫🌍)
- Timer em tempo real + contador de jogadas
- Efeito de flip 3D nas cartas
- Tela de vitória com placar

### 📝 Quiz de Amor
- **5 perguntas** sobre a história de Kauê & Gyovanna
- Feedback visual imediato (verde/vermelho)
- Barra de progresso animada
- Mensagem personalizada por pontuação

## 🚀 Deploy no GitHub Pages

```bash
# 1. Clone ou fork este repositório
git clone https://github.com/SEU_USUARIO/SEU_REPO.git
cd SEU_REPO

# 2. O index.html já está pronto com tudo embutido
# (fotos em base64, sem dependências externas)

# 3. Faça push
git add .
git commit -m "💕 Site romântico para Gyovanna"
git push origin main

# 4. Ative o GitHub Pages
# Vá em: Settings → Pages → Source: Deploy from branch → Branch: main → Save

# 5. Acesse:
# https://SEU_USUARIO.github.io/SEU_REPO
```

## 🛠️ Tecnologias

```
HTML5        → Estrutura semântica completa
CSS3         → Custom Properties, Animations, Glassmorphism, Grid, Flexbox
JavaScript   → Vanilla JS puro, Canvas API, Web Animations API
Google Fonts → Cormorant Garamond + Plus Jakarta Sans
Spotify      → Embed iframe integrado
```

**Zero dependências externas** (exceto Google Fonts e Spotify embed) — tudo roda offline!

## 📐 Arquitetura

```
/
├── index.html          # Arquivo único com tudo embutido
├── images/             # Pasta para manter compatibilidade
│   └── .gitkeep
└── README.md           # Este arquivo
```

O site é um **Single File Application** — todas as fotos estão em base64 dentro do HTML, não há necessidade de servidor, banco de dados ou build step.
