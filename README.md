## 🎮 Os 10 Minigames

### 🔤 1. Wordle — Palavra Secreta
- Palavra de **4 letras** relacionada ao amor de Kauê por Gyovanna
- 6 tentativas, feedback por cores (verde = certo, amarelo = errado lugar, cinza = não existe)
- Teclado virtual + teclado físico suportado
- Animação de flip 3D nas tiles

### 🃏 2. Jogo da Memória
- **16 cartas**, 8 pares temáticos (❤️💕⭐🌙🎵🌹💫🌍)
- Timer em tempo real + contador de jogadas
- Efeito de flip 3D nas cartas
- Tela de vitória com placar

### 📝 3. Quiz de Amor
- **5 perguntas** sobre a história de Kauê & Gyovanna
- Feedback visual imediato (verde/vermelho)
- Barra de progresso animada
- Mensagem personalizada por pontuação

### ❌⭕ 4. Jogo da Velha
- Clássico jogo da velha para dois jogadores
- Interface clean e responsiva
- Reinício automático

### 🎲 5. Dado do Amor
- Dado virtual com **frases românticas especiais**
- Animação 3D ao girar
- Resultados únicos a cada jogada

### 💕 6. Corações Flutuantes
- Clique para adicionar corações que flutuam pela tela
- Contador de corações criados
- Animações suaves e relaxantes

### ⭐ 7. Estrelas Cadentes
- Crie estrelas cadentes com cliques
- Efeitos visuais brilhantes
- Animação de queda suave

### 🎰 8. Roda da Fortuna
- Gire a roda e descubra **frases do destino**
- 8 frases românticas diferentes
- Animação de rotação suave

### 💘 9. Desafio do Amor
- Desafio especial de escolhas românticas
- Perguntas sobre preferências do casal
- Feedbacks divertidos e amorosos

Dê **double click** nos nomes "Kauê & Gyovanna" no título.
> Um segredo especial sobre o casal será revelado!

### 🖱️ Segredo #4 — Múltiplos Cliques no Logo
Clique **3 vezes** no logo/anel do hero.
> Surpresas diferentes a cada sequência de cliques!

## 🚀 Deploy no GitHub Pages

```bash
# 1. Clone ou fork este repositório
git clone https://github.com/SEU_USUARIO/SEU_REPO.git
cd SEU_REPO

# 2. O index.html já está pronto com tudo embutido
# (fotos em base64, sem dependências externas)

# 3. Faça push
git add .
git commit -m "💕 Site romântico para Gyovanna"
git push origin main

# 4. Ative o GitHub Pages
# Vá em: Settings → Pages → Source: Deploy from branch → Branch: main → Save

# 5. Acesse:
# https://SEU_USUARIO.github.io/SEU_REPO
```
